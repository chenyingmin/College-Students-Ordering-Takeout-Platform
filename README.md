项目概述：
本项目为大学生点餐外卖平台，聚焦校园专属场景，核心服务于本校在校大学生、宿舍学生及教职工，不涉及校外配送，支持电脑端与手机端网页响应式浏览，主打校内自取与宿舍配送，贴合校园生活需求。
平台核心有四大功能：商家展示可浏览校园商户信息、搜索筛选并查看评分评价；菜品点餐支持菜单浏览、加购下单、备注，新增拼单与菜品收藏功能；订单管理可查看订单状态、接收提醒、评价及申请售后退款；用户中心支持个人信息管理、校园认证及消费统计，杜绝校外人员使用。
技术选型适配大学生能力，采用 HTML+CSS+JavaScript 搭配 Bootstrap 开发前端，Spring Boot（Java）开发后端，MySQL 存储数据，结合 Git+GitHub 实现 4 人协作，运行环境简单易部署，解决校园点餐不便、拼单麻烦等问题。

功能介绍：
商家展示：展示校园食堂、奶茶店、小吃店基本信息（头像、简介、营业时间、配送范围/自取时间），支持搜索筛选、热门商家排序，新增商家评分与评价功能（用户可对商家服务、菜品质量打分留言，帮助其他同学选择）。
菜品点餐：浏览商家菜单、菜品详情（价格、图片、简介、销量），支持加购、减购、下单、备注（如少糖、多辣），新增拼单功能（同宿舍/同楼层同学可拼单下单，降低配送成本、提高下单效率）和菜品收藏功能（收藏常点菜品，下次下单直接选择）。
订单管理：用户查看个人订单列表，区分待支付、待取餐、已完成、已取消状态，支持查看订单详情，新增订单提醒（取餐提醒、订单状态变更推送）、订单评价功能（下单后可对菜品口感、配送时效评价，联动商家评分），以及退款/售后功能（用户可针对未取餐、菜品不符、漏送等情况发起退款申请，填写退款原因并上传凭证，商家可审核退款申请，审核通过后完成退款，用户可实时查看退款进度）。
用户中心：个人信息编辑、收货地址（宿舍号）管理、历史订单查询、密码修改，新增校园认证（绑定学号/教职工号，杜绝校外人员使用）和消费统计功能（按月/按周统计点餐消费，方便用户合理规划开支）。
